Jessica Roses Portfolio Wireframe Guide

Home Wireframe

Top selected artworks will be the content of the home page. Each work will be proportionally put into a grid.

Work Overview Wireframe

Each selected type of artwork “Illustration”, “Painting”, “Graphic Design” will show an overview of all the content in that category. The actual layout will be the same as the home page. When you click on an image it will be enlarged on a seperate page.

Individual Work Wireframe

When you click on an image it will bring you to a seperate individual page that has more information about the piece. There will be buttons allowing the user to see the next piece, previous piece, or go back to the overview of all the artwork in a certain category.

About Wireframe

The about page will also include my contact information. There will be links to my artwork social media on this page.
